/users


Creates a new user with the provided username, email, password, and birthday.

Parameters:
Name Type Description
req.body.Username string The username of the user to be created.
req.body.email string The email address of the user to be created.
req.body.Password string The password of the user to be created.
req.body.birthday string The birthday of the user to be created. Formatted as yyyy-mm-dd.
