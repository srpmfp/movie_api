Module: DELETE /users/:username/movies/:movieId

Removes a movie from the user's list of favorite movies.
Parameters:
Name Type Description
username string The username of the user whose movie list is being updated.
movieId string The ID of the movie to be removed from the user's list.
Source:

Throws:
If there is an error during the update or if the user does not exist.
