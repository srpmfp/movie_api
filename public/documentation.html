Movie API Documentation

This API is to provide users access to info on movies, their directors, and genres..no

Users

POST /users | create new user
POST /users/:id/:movieTitle | add movie to users favorite
GET /users | get list of all users and their favorite movies
PUT /users/:id | update users information
DELETE /users/:id/:movieTitle || remove a movie from user's favorite
DELETE /user/:id | remove user

Movies

POST /movies | add new movies to the list
GET /movies/:title | selects a movie by title and retrieves its information
GET /movies/genre/:genreName | displays information about the genre
GET /movies/director/:dirName | displays information about the director
