Movie API Documentation

This API is to provide users access to info on movies, their directors, and genres.
use the links below to find formatting, req.body requirements, and resposes from the API

Login

POST /login | log in user

Users

POST /users | create new user
POST /users/:username | add movie to users favorite
PUT /users/:id | update users information
DELETE /users/:id/movies/:movieId | remove a movie from user's favorite
DELETE /user/:id | remove user

Movies

PUT /movies/:director/ | Update director bio and info
PUT /movies/:title | Update the description of the movies
GET /movies/:title | selects a movie by title and retrieves its information
GET /movies/genre/:genreName | returns information about the genre
GET /movies/director/:dirName |returns information about the director
